# Night_Adventure
a fun cat survival game made with html and js ;)

* Night Cat Adventure
A browser-based arcade game built using HTML5 Canvas and Vanilla JavaScript.
The player controls a cat that collects items, avoids hazards, and fights a boss under a dynamic night-themed background.

* Gameplay Summary
Move the cat horizontally.
Collect 🐟 fish to increase score.
Avoid 🍬 candy (causes Game Over).
Collect ⭐ shield for temporary immunity.
Progress through levels.
Defeat boss at Level 3.
High score persists using LocalStorage.

🛠 Tech Stack
HTML5
CSS3
JavaScript (ES6)
Canvas API
LocalStorage API
